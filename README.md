# 💎 I'm Rich - Flutter App

A simple Flutter app that proudly displays a gem 💎 — because you're rich in spirit, knowledge, or maybe even money 😄.

---

## 🏁 Getting Started

This project is a basic Flutter starter app, great for beginners learning UI building, hot reload, and app structure.

### ✅ Requirements

- Flutter SDK: [Install Flutter](https://docs.flutter.dev/get-started/install)
- Dart SDK (comes with Flutter)
- VS Code or Android Studio
- Emulator or a physical device

---

## 🚀 How to Run

1. Clone the repo:

```bash
git clone https://github.com/your-username/im_rich.git
cd im_rich
````

2. Install dependencies:

```bash
flutter pub get
```

3. Run the app:

```bash
flutter run
```
